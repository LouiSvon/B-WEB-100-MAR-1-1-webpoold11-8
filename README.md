# Web Pool D11 - PHP et HTML

## Description

Ce depot contient des exercices PHP generant ou manipulant des elements de page web.

## Objectif

L'objectif est de pratiquer PHP dans un contexte web : fonctions retournant du HTML, pages d'exemple et generation de fragments reutilisables.

## Stack / technologies

- PHP
- HTML

## Structure

```text
.
└── day11/
    ├── index01.php
    ├── index02.php
    ├── index03.php
    ├── task01.php
    ├── task02.php
    └── task03.php
```

## Fonctionnalites principales

- Generation d'un menu HTML.
- Fichiers `index` pour tester les exercices.
- Fonctions PHP separees par tache.

## Lancement

Depuis le dossier du depot, lancer un serveur PHP local :

```bash
php -S localhost:8000
```

Puis ouvrir une page de test :

```text
http://localhost:8000/day11/index01.php
```

## Remarques

Les fichiers `taskXX.php` peuvent aussi etre inclus dans d'autres scripts PHP.
